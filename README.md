# Natural Language Processing 

Developed by [Jiayi Chen](https://github.com/jia-yi-chen) for [CS 6501 Natural Language Processing](https://github.com/jiyfeng/uva-nlp-class).

## Requirements

* python 3
* pytorch 1.0+

## Topics included

* Bag of Words
* Perceptron & Averaged Perceptron
* Logistic Regression
* POS Tagging
* Hidden Markov Model
* LSTM Language Model
